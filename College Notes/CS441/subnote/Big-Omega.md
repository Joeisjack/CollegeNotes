Big-Omega Example #1
Show that 10n log(n) is Omega(n). Prove this

Remember:
c g(n) <= f(n) for every n >= n_0

c = 10, g(n) = n, f(n) = 10 * n * log(n), n_0 = 2

10 * n <= 10 * n * log n forall n >= 2
n <= n log n for all n >= 2 ---> TRUE
